## Usage

### Interactive Converter

Open `reference.html` in any browser. Type text, select a register, toggle modifiers — get ACEDIT output. Click any glyph to copy it. Zero dependencies, pure HTML/CSS/JS.

### Programmatic Access

Load `acedit-core.json` in any language:

```javascript
const acedit = JSON.parse(fs.readFileSync('acedit-core.json'));
const ultra = acedit.registers.ULTRA;

// Convert a character
function toACEDIT(char, register) {
  const reg = acedit.registers[register];
  if (char >= 'A' && char <= 'Z') return reg.upper[char];
  if (char >= 'a' && char <= 'z') return reg.lower[char];
  if (char >= '0' && char <= '9') return reg.digits[char];
  return acedit.punctuation[char] || char;
}
```
